###  正浮点数

```
^\d+(\.\d+)?$
```

### 正数(正整数+正浮点数)

```
^[1-9]\d+(\.\d+)?$
```

```
^((0{1}\.\d{1,n})|([1-9]\d*\.{1}\d{1,n})|([1-9]+\d*)|0)$
n 表示小数点后保留位数
```

### 二进制

```
^[01]+$
```

### 八进制

```
^[0-7]+$
```

### 十六进制

```
^#?([a-f0-9]{6}|[a-f0-9]{3})$
```

### 数字正则匹配

```
(^[1-9]\d+(\.\d+)?$)|(^[1-9]\d+(\.\d+)?$)|(^((0{1}\.\d{1,n})|([1-9]\d*\.{1}\d{1,n})|([1-9]+\d*)|0)$)|(^[01]+$)|(^[0-7]+$)|(^#?([a-f0-9]{6}|[a-f0-9]{3})$)
```
